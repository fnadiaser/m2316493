# 1. Назначение
Методика определяет состав KPI, правила расчета план против факт и требования к пояснениям отклонений.

# 2. Состав KPI (пример для MVP)
- Объем переработки, т
- Извлечение металла, проценты
- Выход готового продукта, т
- Простой оборудования, часы
- Доля корректных входных данных, проценты

# 3. Расчет отклонений
## 3.1 Абсолютное отклонение
Отклонение_абс = Факт - План

## 3.2 Относительное отклонение
Отклонение_проц = (Факт - План) / План * 100 процентов

Правило: если План равен нулю, процент не рассчитывается и требуется пояснение причины.

# 4. Классификация причин отклонений
Причины должны относиться к одной или нескольким группам:
- Сырье: дефицит, задержка поставок, несоответствие качества.
- Мощности: ограничение производительности, узкие места, отсутствие персонала.
- Ремонты: аварийные, плановые, наладка.
- Логистика: задержка поставки, транспорт, склад.
- Качество данных: расхождения источников, устаревшая выгрузка, неверный формат.

# 5. Требования к пояснению отклонений
Пояснение обязательно включает:
1) Период, участок, показатель.
2) План, факт, отклонение.
3) Причины (не более 5), с ранжированием по вероятности.
4) Подтверждающие аргументы: события, ограничения, ссылки на регламент.
5) Влияние на выполнение плана и прогноз.
6) Корректирующие меры: действие, ответственный, срок.

# 6. Правило достоверности
Если нет данных для подтверждения, формулировка должна быть условной, а также задаются уточняющие вопросы.
